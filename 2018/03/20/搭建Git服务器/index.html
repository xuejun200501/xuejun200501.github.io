环境

可能错误

1、SSH连接远程服务器（Linux-Centos）以及出现问题解决

解决办法：rm /.ssh/knownhosts

2、出现“Could not read from remote repository.”

解决办法：查看服务器的project路径，如上面，服务器的路径为/home/git/project.git，并非/home/project.git

3、出现要求输入密码

解决办法：这种报错一定是ssh密钥问题，所以直接查看服务器的公钥与客户端的是否一致
ps：可能存在客户端的用户名为中文，且乱码的情况，所以尽量用英文做用户名，以防不可预测的错误
