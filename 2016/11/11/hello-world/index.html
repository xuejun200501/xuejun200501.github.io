水银泻地 无懈可击

Hello World

前言

一直以来，都梦想着建立自己的博客，但人天性就是懒惰的，直到某一天，发现不知不觉中，从第一次接触谭浩强版《c语言程序设计》到现在，弹指之间整整十年，不得不感慨，人生天地之间，若白驹之过欲，忽然而已… …

对个人

曾经的我，不可一世的认为干好老板交代的事情，用技术做出牛叉的产品，充分证明自己的价值便可，也许这都是很多职业编码人的通病，或者是很多人对程序员很傲娇的切实体会，确实很多的时候，程序员可以在自己的编码中解决所有生活中不能解决的问题，他们可以将各种优越感在编码中彰显地淋漓尽致，对于他们来说这是一片净土，抛离所有烦恼，在编码中任意驰骋… …
然，在现在多元化的社会，很多时候，很多情况，人不可能逃离其他人而独自生活，当你没钱生活时，需要向身边信任的人需求生计问题，当你遇到技术问题时，需要寻找比你更专业的人寻求帮助，当你遇到烦恼时，需要他人帮你剖析、倾听… …
过去的一年，对个人应该来说，截止今天为止，应该说是工作以来体会最深的一年，也许成长总是要付出种种代价，未来的我也会少些愤怒、少些抱怨、少些高冷、多些从容、多些谦虚、多些微笑… …

对他人

已经过去的曾经，很多人找过我帮助，修下电脑，装个系统，分手了一起扯扯淡等等，很多时候，大家会很客气，很礼节性的请吃饭、喝coffee等等，大多时候都被我直接回绝，当然一定时间也为了表现我没那么傲娇或者其他情况，我也接受。对于他人，我只想表明一个观点，其实很多时候我只是举手之劳而已，何足挂齿，也没有傲娇到吃个饭喝个水都不愿意。对于我来讲，一定程度，一方面我很开心能得到你的信任共同面对问题，另一方面其实我也可以学到很多东西，或者有了更好的交际圈，我认为这是无形的资产，如同我毕业了仍然在深大住了三年，如果按照现在我一个月房租1600元／月计算，三年下来我应该给予别人好几万，而事实上好像我并未花什么钱；今年找工作，腾讯，yy，大疆之类的内推也是多年的同学朋友在知悉我准备换工作情况下自愿内推，等等这些并非用钱、用固定的利益可以衡量，或许真诚更重要… …

对工作

好好工作… …

对对象

虽然在编码过程中，作为developer的我可以new各种的object，可以根据需求进行各种的属性设置，然而生活中的对象并不是可以随便new一个。
当今社会，人们的生活水平已经到了小康水平，年轻人之间讲究的更多的是思想的融合。一个美女遇到一个帅哥，固然大家都开心，随着时间的推移，新鲜感逐渐退去后，可能更多的是思想的交汇，彼此之间能不能一起happy，一起share自己的不快，一起促进彼此进步，能否相互理解，彼此信任可能才是关键，其实有没有相同的价值观，共同的人生观的或许最重要… …

对创业

2010年的初夏，凭着满腔热血，写了人生第一次商业计划书，筹建着自己的梦想之路，其实没有核心竞争力，缺乏市场判断，思维混乱等等，可想而知，结果是非常的凄惨。当然那个阶段也是我人生最宝贵的经历，记得临开业的一个月日子里，基本上都是早上5点睡觉，7点便起床，仅仅二三个小时的睡眠时间。这也让我懂得了创业的痛苦，当然也正是这种经历，让我在每份工作更是竭尽全力。所以未来的二三年的时间内，或者未来的很长时间内，如果没有想好的idea，没有核心的technology，不会轻易startup。当然大家可以一起畅谈人生，讲讲故事了，如果不是童话里的，说不定也可以碰出那么一撮火花… …

对吃喝

很多人反应我吃饭很傲娇，一般场合都不愿意去吃，或者去了不怎么吃，非也。曾经在某公司时，老板发现他的某同事有个特点，带某同事去吃饭，由于同事从大老远过来，所以特意好酒好肉招待，但是期间发现该同事很少喝酒，本以为不喝，也没有太在意，一天，又在一起喝酒，发现此同事喝酒都是一口干，观察得出结论，此同事不是不喝酒，好酒干一口，劣酒抿一口，所以认为我也是如此，其实不然，我喝酒特看气氛，要么很happy，要么很忧伤才能喝得下，否则肠胃极其不舒服，另外由于肠胃不是很好，或者个人习惯问题，在外面比较喜欢看着别人吃，而自己并不好吃特吃，也并非是傲娇不吃什么云云，如果换成家常菜自然不会客气……

对技术

十几年初次接触命令行，感觉计算机技术很厉害，现在依旧是，人生最不后悔的事情，就是大学学的专业与自己毕业后的工作一直是计算机技术，即使下班后，研究下前沿技术依旧是充满斗志，幸福满满，当然大学对经济学、金融学也有极大兴趣… …

对扯淡

很多时候，很多人，都一致认同一个观念，就是你们搞技术的人是不是很呆，很难沟通云云。一定程度上，对一些人确实是这样，高中以前，即使没搞技术好像我也是如此，由于自己的不够优秀，所以不敢太多交流。但是大学班长及学生社团经历彻底改变了我，当我不得不去面对很多问题，不得不去解决我所面对的问题时，我学会了沟通，尤其是在深大这种高度开放、自由的氛围下，在专业课之外的各种公选课，让我学会了从不同的角度，用不同的思维思考问题的能力，既能客观的阐述自己的观点，又能让人心悦诚服。记得今年年前跟某VC讲故事时，我竟然从早上10点多到20点半共计10个多小时时间里阐述自己的价值观、人生观，中间只吃了一个晚饭，上了几次厕所，中饭都木有吃，如果不是赶时间回深圳，时间持续更久，虽然梦想着百万级的投资，但遗憾是，并未融资成功。故，对个人来讲，具备一定扯淡能力，但讲故事本身还是欠佳。当然这些都是过去，未来的道路上，这些只是基石而已，今后还有很长的路，也有很多的东西需要学习……

对社交

不是很喜欢利用SNS来促进感情，尤其比较反感平时聚餐或者难得聚聚时，每个人时刻看着手机，好像很多人找、很忙一样，个人较喜欢面对面的交流，探讨。过去经常被骂有些孤傲，不回或者没及时回复他人信息，其实之前很多时候我很少时刻拿着手机或者正在编码之中，思考代码结构之时，虽然看了信息，但很容易忽视掉，so如果真有事可直接电话或者多次信息，当然现在会尽量避免这些问题。当然如果未来我能开发出一款大家每天都乐意打开的app，我还是希望多拿着手机，哈哈哈… …

对过去

过去的过去，开心不开心，努力不努力，都已成为过去式，既然无法改变，为何要去怀念没有意义的过去… …

对现在

努力工作，提高自己技术能力、业务能力的同时，将生活好好进行下去。没事坐在飘窗前，手持一本文学著作，时而端起coffee，时而眺望远方，憧憬未来美好生活

对未来

不懈努力后，有了自己的家庭，经济自由，和自己的家人，行走世界各地；如果某一天走不动了，可以回到田园，过着采菊东篱下的桃园生活

对生活

努力生活… …

对有些话

人只要脾气好，凡事就会好
