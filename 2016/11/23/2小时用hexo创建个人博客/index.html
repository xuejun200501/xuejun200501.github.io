2小时用hexo创建个人博客

从准备搭建自己的博客，到完整技术实现，应该有二个多星期技术调查，看着各种各样的长篇介绍、概括、总结等等，看得人都晕了，为了其他有意愿搭建自己的博客的同学，少走弯路，故写下此篇，希望对各位有帮助

由于自己一直用的是Mac，所以此篇主要以Mac环境为主，其他Linux、centos、ubuntu等应该差不太多，当然如果有需求，可联系我，我可在windows环境再完全配置下，以便帮助大家

一、本机环境配置（必要）

  • Node.js
  • Git

    注:
    1 、对于git，Mac直接有自带
    2、node.js我是直接去官方下载包安装
    ps：貌似某些资料说用命令brew命令，具体没有试过，等有时间我再测试吧，如果你非要装b或者担心出问题，那也可以参考此资料；node官网有二个版本，一个LTS，current，个人认为没有关系，从字面意思理解LTS:Long Term Support就是稳定点，后者可能由于版本新，存在一些未知bug而已，我选择的是LTS

二、Github环境配置

1、毫无疑问，注册一个GitHub帐号
2、建立一个仓库，切记Repository name的格式，如：xuejun01.github.io，后面一定为.github.io格式
3、git clone

4、设置git的name和email

5、生成ssh密钥

  • 没有生成密钥

  • 已生成密钥

注：pbcopy是我感觉Mac下比较帅的命令了，上面命令的意思是将id_rsa.pub里面的公钥复制到剪贴板中，想对应的还有pbpaste_。当然如果这样你都还不会生成ssh密钥，那看下官方文档

6、在GitHub中粘贴公钥

四、初始化hexo

1、找个自己准备放博客的目录，如我放博客的位置

3、进入blog，且初始化hexo
